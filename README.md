# Signup Automation – QA Task

This repository contains an automated signup flow for:
https://authorized-partner.vercel.app/

## Tech Stack
- Python
- Playwright (Sync API)

## Files
- signup_automation.py – automation script
- README.md – execution steps
- Test_report.pdf – test cases
- demo_video.mp4 – execution demo

## How to Run
1. Install Python 
2. Install dependencies:
   pip install playwright
   playwright install
3. Run:
   python signup_automation.py# signup-automation
